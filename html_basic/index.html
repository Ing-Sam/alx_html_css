HTML- HyperText Markup Language, is a markup language that tells a web browser how to structure web pages

HTML consists of a series of elements which you use to enclose, wrap or mark up different parts of contents to make it appear or act in a certain way.

Element

An element is a part of a webpage in XML or HTML. It may contain a data item or a chunk of text or an image or perhaps nothing.

Tag(< >) begin or end an element in a source code.

Void Elements

Not all elements follow the pattern of an opening tag, content, and a closing tag. Some elements consists of a single tag, which is typically used to insert/embed something in the documents.

Eg. <img> elements embeds an image file onto a page.
